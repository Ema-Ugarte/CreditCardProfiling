# Estudio de perfiles en solicitudes de tarjeta de crédito

+ ## Descripción:
**El departamento de tarjetas de crédito de un banco es uno de los principales en utilizar la ciencia de datos. Siendo uno de sus principales objetivos: La captación de nuevos clientes. El banco tiene en cuenta que otorgar tarjetas de crédito sin realizar una investigación adecuada ni evaluar la solvencia de los solicitantes es bastante arriesgado. Es por ello que el banco utiliza un sistema de puntuación crediticia que evalúa la solvencia del solicitante mediante datos internos y externos. Este modelo no solo determina la aprobación, sino que también define el límite de crédito y el nivel de riesgo, permitiendo ajustar las decisiones a la estrategia actual de la institución.**
